# TDD - 2026-03-27: 翻译卡片高度自适应与 LLM 配置检查优化

## 一、需求背景

1. **翻译卡片弹窗太小** - 用户希望卡片高度能自适应内容长度，并有一个合理的上限
2. **首次启动 LLM 配置引导** - 用户未配置 LLM 时应弹出配置弹窗，默认值应为空但提供 placeholder 提示

## 二、技术实现

### 2.1 翻译卡片高度自适应

**修改文件**: `ReflectPDF/Views/TranslationBubble.swift`

**原有问题**:
- 卡片使用固定高度约束，内容较多时显示受限
- `ViewThatFits` 虽然支持滚动，但外层缺少最大高度限制

**解决方案**:
```swift
// 在 card 视图上添加 maxHeight 约束
.frame(width: 380)
.frame(maxHeight: 680) // Upper bound for long content
```

**技术要点**:
- SwiftUI 的 `frame(maxHeight:)` 允许视图根据内容自适应高度，同时设置上限
- 原有的 `ViewThatFits` + `ScrollView` 方案保持不变，用于处理超长内容
- 680px 的上限既能展示足够内容，又不会占据过多屏幕空间

### 2.2 LLM 配置检查优化

**修改文件**: `ReflectPDF/Views/ContentView.swift`

**原有问题**:
- 只检查 `baseURL` 和 `apiKey`，未检查 `model`

**解决方案**:
```swift
@AppStorage("llm_model") private var model = ""

.onAppear {
    let storedKey = KeychainService.load(key: "llm_api_key") ?? ""
    if baseURL.isEmpty || storedKey.isEmpty || model.isEmpty {
        showSetupSheet = true
    }
}
.sheet(isPresented: $showSetupSheet) {
    SettingsView(onDismiss: { showSetupSheet = false })
        .environmentObject(appState)
}
```

**技术要点**:
1. `@AppStorage` 属性包装器用于持久化存储用户偏好
2. 使用 `onDismiss` 闭包正确关闭 sheet（SettingsView 已支持此模式）

### 2.3 SettingsView 按钮布局

**修改文件**: `ReflectPDF/Views/SettingsView.swift`

**实现**:
- 作为设置弹窗显示时，提供"稍后设置"按钮
- 按钮布局：左侧留白 → 已保存标记 → 稍后设置 → 保存设置（主按钮）
- 使用 `.borderedProminent` 样式突出保存按钮

```swift
HStack {
    Spacer()

    if showSavedBadge {
        Label("已保存", systemImage: "checkmark.circle.fill")
            .foregroundStyle(.green)
    }

    if let dismiss = onDismiss {
        Button("稍后设置") {
            dismiss()
        }
    }

    Button("保存设置") {
        saveSettings()
        onDismiss?()
    }
    .buttonStyle(.borderedProminent)
}
```

### 2.4 SettingsView 默认值与 Placeholder

**文件**: `ReflectPDF/Views/SettingsView.swift`

**当前实现**:
```swift
@AppStorage("llm_base_url") private var baseURL = ""
@AppStorage("llm_model") private var model = ""

TextField("例：https://api.openai.com/v1", text: $baseURL)
TextField("例：gpt-4o-mini", text: $model)
```

- 默认值为空字符串 `""`
- placeholder 提供示例值引导用户输入

## 三、验证要点

1. **卡片高度**:
   - 短内容：卡片紧凑显示
   - 长内容：卡片扩展至最大高度后内部滚动
   - 交互：拖拽、按钮点击不受影响

2. **LLM 配置引导**:
   - 首次启动（无配置）：弹出配置弹窗
   - 点击"稍后设置"：弹窗关闭，下次启动仍提示
   - 部分配置（如只填了 URL）：仍提示完善配置
